# 🎬 TrailerFlix

TrailerFlix is a Netflix-inspired web application that allows users to browse and watch trending movie trailers in a modern and responsive interface.

---

## 🚀 Features

- 🎥 Watch movie trailers instantly
- 🔥 Browse trending and popular movies
- 📱 Fully responsive Netflix-style UI
- ⚡ Fast and smooth user experience
- 🌐 API-based dynamic movie content
- 🎨 Clean and modern design

---

## 🛠️ Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- Movie API / TMDB API
- Vite / Create React App

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/trailerflix.git
```

Go to project folder:

```bash
cd trailerflix
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
